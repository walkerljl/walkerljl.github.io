# 一、下载和安装
 - 进入地址redis下载地址：https://redis.io/download
    下载最新稳定版本（如：redis-3.2.6.tar.gz）
 - 解压：tar -zxvf redis-3.2.6.tar.gz
 - 进入解压后的目录：redis-3.2.6
 - 编译：make
 
# 二、启动服务
- 进入服务目录：redis-3.2.6/src
- 执行命令：./redis-server

# 三、控制台
- 进入服务目录：redis-3.2.6/src
- 执行命令：./redis-cli
